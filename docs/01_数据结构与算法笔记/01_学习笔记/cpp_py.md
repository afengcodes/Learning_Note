c++

**判断大小写**
islower(char)
isupper(char)
**判断是否是字母**
isalpha(char)  可代替  isupper( ch ) || islower( ch )
    - 判断字符ch是否为英文字母，若为英文字母，返回非0（小写字母为2，大写字母为1）。若不是字母，返回0。
**判断数字**
isdigit(char)

// 其他
